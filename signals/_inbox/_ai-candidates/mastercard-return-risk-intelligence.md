# Mastercard Return Risk Intelligence

> Giving merchants a cross-network fraud signal at the moment a return is requested, before a refund is issued.

## Anti-hype marker

> Return Risk Intelligence uses AI to analyze current and historic anonymized Mastercard transactions, specifically returns, refunds and dispute records. It identifies patterns and behaviors that correlate with reported fraud or chargeback events, using this intelligence to generate scores that help merchants predict which returns are more likely to be fraudulent.
